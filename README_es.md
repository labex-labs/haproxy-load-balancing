Este módulo cubre la Instalación de HAProxy, Balanceo de Carga HTTP, Panel de Control y Estadísticas, Balanceo a Nivel TCP, ACLs y Enrutamiento Inteligente. Dominará estas habilidades esenciales de Linux a través de laboratorios prácticos y desafíos del mundo real.
